# SiListen

SiListen 是一个使用 Kotlin + Jetpack Compose 开发的 Android 音乐播放器。项目目标是做一个简洁、沉浸、接近 Apple Music 观感的移动端音乐应用，同时保留 Android 平台的本地音乐、桌面歌词、状态栏歌词、媒体通知控制和自定义音源能力。

项目当前仍处于持续完善阶段，界面、音源策略和播放体验会继续迭代。

## 项目特点

- Apple Music 风格播放页、歌词页、迷你播放器和沉浸式界面。
- 网易云账号登录、歌单同步、红心收藏、评论展示与加载更多。
- 歌单、专辑、歌手、单曲搜索，并支持多平台结果聚合。
- 自定义 LX 风格音源脚本，用于解析播放地址，不需要额外启动本地 `3000` API 服务。
- 本地音乐扫描、本地歌单库、本地歌单创建与歌曲加入。
- Apple Music 风格歌词、逐字高亮歌词、翻译/罗马音展示。
- 通知栏媒体控制器、桌面歌词、状态栏歌词、播放模式和定时停止播放。
- 端内直连网易云接口，自实现 WeAPI/EAPI 加密，不依赖 Node.js 网关。

## 功能概览

### 音乐浏览

- 首页推荐与今日播放。
- 音乐库与本地音乐入口。
- 搜索页支持单曲、歌单、专辑、歌手分类。
- 歌手页支持详情、热门、单曲、专辑切换。
- 歌单详情支持当前歌单内搜索。

### 播放体验

- Media3 ExoPlayer 播放内核。
- 播放队列、上一首、下一首、播放/暂停。
- 顺序播放、单曲循环、随机播放等播放模式。
- 定时停止播放，可选择等待当前歌曲结束。
- 通知栏媒体控制器，支持喜欢、上一首、播放/暂停、下一首、桌面歌词开关。

### 歌词体验

- Apple Music 风格全屏歌词页。
- 当前行高亮、平滑滚动、逐字填充。
- 翻译和罗马音同步展示。
- 歌词为空时支持手动刷新。
- 桌面歌词和状态栏歌词可单独开关与调整。

### 账号与收藏

- 网易云账号登录。
- 同步用户歌单、喜欢的音乐、云盘、播客和每日推荐等内容。
- 歌曲红心收藏与取消收藏。
- 自己创建的网易云歌单保持只读收藏状态。
- 收藏他人歌单时支持取消收藏确认。
- 本地歌单库支持创建、长按删除和加入歌曲。

## 音源策略

SiListen 默认以网易云身份作为账号、歌单、收藏和评论能力的主线。播放地址可以通过用户导入的自定义源脚本解析，用于处理部分歌曲无法直接播放、VIP、灰色歌曲或无版权的场景。

推荐理解方式：

- 账号、歌单、红心、评论：优先使用网易云身份。
- 实际播放地址：可以走自定义音源脚本兜底解析。
- 搜索、歌词、封面：根据应用内音源策略匹配，并在失败时尝试兜底。

从当前版本开始，应用不再需要用户手动启动本地 `NeteaseCloudMusicApi` 或 `3000` 端口服务。

## 本地音乐与本地歌单

本地音乐扫描依赖系统音频读取权限：

- Android 13 及以上：`READ_MEDIA_AUDIO`
- Android 12 及以下：`READ_EXTERNAL_STORAGE`

扫描完成后，音乐库会显示“查看全部本地歌曲”入口，而不是直接铺开所有本地歌曲。用户可以在“本地歌单库”中新建歌单，并通过歌曲右侧菜单的“加入歌单”把歌曲加入本地歌单。

本地歌单保存在应用私有的 SharedPreferences 中，保存的是歌曲快照，因此可以混合保存本地歌曲和在线歌曲。

## 构建方式

推荐使用 Android Studio 打开项目根目录：

```text
D:\SiListen
```

命令行构建调试包：

```powershell
$env:JAVA_HOME='E:\Program Files\Android\Android Studio\jbr'
.\gradlew.bat :app:assembleDebug --console=plain
```

构建成功后，调试 APK 通常位于：

```text
app\build\outputs\apk\debug\app-debug.apk
```

## 权限说明

- 网络权限：访问音乐接口、评论、歌词、封面和自定义音源脚本。
- 通知权限：显示媒体播放通知和通知栏控制器。
- 音频读取权限：扫描本地音乐。
- 悬浮窗权限：显示桌面歌词。
- 前台服务权限：保证播放和歌词悬浮窗服务稳定运行。
