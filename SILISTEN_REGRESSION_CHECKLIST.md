# SiListen 0.2 真实设备回归清单

> 推送前由本人实测确认。每轮只记录当前 APK 的真实设备表现，不用猜。

## 1. 启动与基础稳定性

- [ ] 冷启动进入应用，无闪退。
- [ ] 按 Home 返回桌面，再点图标回到应用，无闪退。
- [ ] 返回桌面后再进应用，不会自动跳回首页打断当前页面。
- [ ] `adb logcat -b crash -d -t 200` 无新增崩溃。

## 2. 播放生命周期

- [ ] 播放一首在线歌曲后，底部迷你播放器出现。
- [ ] 播放中按 Home 返回桌面，音乐继续播放。
- [ ] 从桌面回到应用，当前歌曲、播放进度、迷你播放器仍然存在。
- [ ] 点击通知栏进入应用，能打开播放页/歌词页。
- [ ] 通知栏上一首、播放/暂停、下一首、关闭都可用。
- [ ] 通知栏爱心可加入/取消“我喜欢的音乐”，状态显示正确。

## 3. 底栏与搜索

- [ ] 底栏点击首页、音乐库、账号不会串页。
- [ ] 底栏左右拖动切页跟手，选中位置正确。
- [ ] 点击搜索按钮展开搜索，再点任意底栏 tab，回到对应 tab，不固定回第一次来源。
- [ ] 搜索页返回关闭后，回到打开搜索前的页面。
- [ ] 搜索结果进入歌手/歌单/专辑详情后，返回先回上一级，再回搜索页。

## 4. 歌单与歌手详情

- [ ] 歌单页歌曲右侧三点菜单可打开。
- [ ] 三点菜单“下一首播放 / 加入歌单 / 查看歌曲评论”可用。
- [ ] 歌手页“详情 / 热门 / 单曲 / 专辑”可以左右滑动切换。
- [ ] 歌手页进入专辑后，返回先回歌手页。

## 5. 播放页与歌词

- [ ] 播放页详情、歌词、队列、评论四个页面可切换。
- [ ] 歌词当前行滚动正常，没有双层歌词重叠。
- [ ] 两行歌词逐字高亮不会在换行瞬间闪回。
- [ ] 翻译/罗马音显示不被圆角裁切。
- [ ] 开启状态栏歌词后，通知栏副标题显示当前歌词；关闭后恢复歌手名。

## 6. 首页与账号

- [ ] “最近播放”显示在“我的歌单”第一行。
- [ ] 最近播放点击后能进入列表并播放歌曲。
- [ ] 登录状态刷新不会清空正在播放。
- [ ] 本地音乐入口、扫描、列表播放可用。
